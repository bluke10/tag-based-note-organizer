# Tag-Based Note Organizer

A simple Java application that lets users create notes with tags and search them using set operations like intersection, union, and difference.

## Features
- Add notes with multiple tags
- Search notes by:
    - All tags (AND / intersection)
    - Any tags (OR / union)
    - Excluding tags (difference)

## Tech
- Java
- Uses HashSet for efficient tag operations

## How to Run
1. Clone the repo
2. Compile:
   javac *.java
3. Run:
   java Main
